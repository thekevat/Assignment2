Computer Networks

The Computer Networks course at IET DAVV provides a strong foundation in networking principles, protocols, and technologies essential for modern communication systems. The course covers both theoretical concepts and practical implementations, preparing students for careers in network engineering, cybersecurity, cloud computing, and software development.

  • Network Models & Architectures – OSI & TCP/IP models, network topologies
  • Data Communication – Transmission media, error detection, and correction
  • Network Layer & Routing – IPv4/IPv6, subnetting, routing algorithms (RIP, OSPF, BGP)
  • Transport Layer – TCP & UDP, congestion control, flow control
  • Application Layer Protocols – HTTP, FTP, DNS, SMTP, SNMP
  • Network Security – Firewalls, encryption, VPN, SSL/TLS
  • Wireless & Mobile Networks – Wi-Fi, 4G/5G, IoT networking
